1. count(value)

Purpose: Counts how many times a specific value appears in the tuple.

Example:

t = (1, 2, 3, 2, 4, 2)
print(t.count(2))  # Output: 3



2. index(value)

Purpose: Returns the index of the first occurrence of a value.



Example:

t = (1, 2, 3, 2, 4, 2)
print(t.index(2))  # Output: 1
